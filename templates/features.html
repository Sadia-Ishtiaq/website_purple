Powerful Features

Advanced AI-powered tools for accurate brain tumor detection and analysis

Multi-Class Classification

Our ResNet50 deep learning model classifies brain MRI scans into four categories:

  • Glioma: Common brain tumor type
  • Meningioma: Usually benign tumors
  • Pituitary: Affecting pituitary gland
  • No Tumor: Normal brain tissue

YOLOv8 Tumor Localization

State-of-the-art object detection to precisely locate tumors:

  • Bounding boxes around detected tumors
  • Confidence scores for each detection
  • Multiple tumor detection in single scan

Confidence Scoring

Each prediction comes with detailed confidence metrics:

  • Probability percentage for each class
  • Multi-class probability distribution
  • Threshold-based recommendations

Comprehensive Reports

Generate detailed PDF reports including:

  • Original MRI image
  • YOLO localized tumor detection
  • Diagnosis and confidence scores
  • Patient information and timestamp
  • Clinical recommendations
